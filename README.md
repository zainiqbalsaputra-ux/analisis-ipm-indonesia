# Analisis IPM Indonesia

Penelitian ini menganalisis faktor-faktor yang mempengaruhi Indeks Pembangunan Manusia (IPM) kabupaten/kota di Indonesia.

## Struktur Proyek

data/raw → Dataset mentah  
data/processed → Dataset panel final  
notebooks/01_data_cleaning → Proses pembersihan data  
notebooks/02_analisis_ipm → Analisis statistik & regresi  

## Variabel
- IPM
- Pengeluaran per kapita
- Rata-rata lama sekolah
- Umur harapan hidup
- Persentase penduduk miskin

## Metode
- Data panel
- Regresi OLS
- Uji asumsi klasik
- Robust standard error (HC3)

## Cara Menjalankan
1. Jalankan notebook data cleaning
2. Jalankan notebook analisis
